Modelo de caixas

Todo elemento visivel esta dentro de uma caixa

  • height: altura
  • width: largura
  • border: é uma linha que circunda o conteudo
  • padding: para deixar a borda distante do conteudo. borda para dentro
  • margin: borda para fora

Tipos de caixa

  • box-level: ocupa a largura total da tela. ex: div,h1, p etc
  • inline-level: pula para o lado, não quebra a linha, ocupa o tamanho que precisa. ex: span,link etc

grouping tags

Funcionam como div para organizar melhor: header- cabeçalho, main- conteúdo principal, footer - final da pagina, section - seções, etc.
