o pai no caso o container é que recebe as configurações flex-direction: row (linha) - é esquerda para direita row-reverse - direita para esquerda flex-direction column (coluna) - de cima para baixo column-reverse - debaixo para cima Eixos: Principal (main-axis) main-start, main-end. Row reverse muda para direita para esquerda. eixo transversal(cross axis) cross-start, cross-end. No row reverse não muda.

flex-wrap: configura o comportamento da capsula (pai), que por padrão vem nowrap (não quebra).

o encolhimento depende do conteudo que existe no blaco

no flex-wrap:wrap ele quebra no cross-axis o ultimo elemneto vai para baixo

wrap-reverse: o envelopamento vai no sentido oposto do cross-axis a caixa filha vai para cima

pode ser utilizado o shorthand: flex-flow= row nowwrap por exemplo

justify-content: alinha de acordo com o main-axis
